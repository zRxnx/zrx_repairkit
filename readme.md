# zRxnx's Simple Repairkit System

This is a simple repairkit system for fivem

<b>Features:</b>
- Item based
- Fancy discord log
- Update checker
- Multiple repairkits support
- With animation
- 0.0 ms on idle
- 0.0 ms while in use

## Requirements
- es_extended
- esx_progressbar

## Installation
- Download script
- Put script in your `resources` directory
- Add `ensure zrx_repairkit` in your `server.cfg`
